# Bosco Global FZ LLC — Maintenance Page

An animated "under maintenance" web page built with React + Vite.

## Features
- Animated spinning gears with pulse rings
- Live progress bar with shimmer effect
- Floating particle animations
- Atmospheric gradient orbs
- Dark theme with golden accents
- Contact card linking to info@bosco.ae

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Getting Started

```bash
pnpm install
pnpm dev
```
